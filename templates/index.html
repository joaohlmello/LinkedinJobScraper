Instructions
  1. Paste one or more LinkedIn job listing URLs into the text area above.
  2. Click the "Extract Information" button.
  3. The script will extract job title and company information from each URL.
  4. Results will be displayed in a table with the original link, company name, company link, job title, and job description.
  5. You can export the results to CSV or Excel format using the buttons above the results table.

Example URL format:

https://www.linkedin.com/jobs/view/4197948497/?eBP=NON_CHARGEABLE_CHANNEL&refId=bkHAVjq057sZNx6xA%2BnKig%3D%3D&trackingId=hSdg4bhpEUyduU4ScMhsgg%3D%3D&trk=flagship3_search_srp_jobs
Nota importante sobre "Tipo de Candidatura"

O LinkedIn exibe diferentes informações dependendo se o usuário está logado ou não:

  • Quando logado no LinkedIn, você pode ver opções como "Easy Apply" ou "Candidatura Simplificada".
  • {% if has_linkedin_credentials %}Este scraper agora tenta acessar o LinkedIn com login para determinar o tipo de candidatura com maior precisão.{% else %}Este scraper está configurado para acessar o LinkedIn sem login, então ele pode mostrar apenas "Apply" mesmo quando a vaga é "Easy Apply".{% endif %}
  • Para vagas do LinkedIn Brasil (.com.br) ou outras regiões da América Latina que mostram "Apply", indicamos como "Apply (possível Easy Apply*)" porque podem ser vagas de "Easy Apply" quando visualizadas com login.
